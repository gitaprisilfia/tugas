Profil Saya

Saya adalah salah satu Mahasiswa di Universitas Halu Oleo, Jurusan Teknik Informatika angkatan 2019. Saya berusia 18 tahun. Saya berasal dari Kabupaten Buton Utara, Provinsi Sulawesi Tenggara. Alasan saya masuk di jurusan teknik informatika adalah karena saya ingin menjadi bagian dari revolusi industri 4.0, serta ingin berkontribusi dalam perkembangan dan pembangunan bagi bangsa dan negara melalui bidang IT.

Hobi Dan Pengalaman Organisasi

Saya memiliki lumayan banyak hobi tetapi memanahlah yang paling saya gemari, saya mulai mengenal olahraga tersebut sejak saya duduk di bangku sekolah dasar sampai saat ini saya masih sering memanah jika ada waktu luang. Saya juga merupakan atlet panahan . Untuk pengalaman organisasi dulu ketika di SMA saya pernah menjadi ketua MPK, sekretaris OSIS, dan menjadi ketua ambalan di gudep sekolah saya.
